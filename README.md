<h1 align='center'>kmz2geotiff</h1>

<p align='center'>A python script to convert .kmz files to GeoTIFF format for use with TAK and other geospacial programs.</p>

## Pre-Requisites / Setup

1. Install [Python](https://www.python.org/downloads/) if not already installed. 
2. Install [GDAL For Python (make sure it matches your version of Windows)](https://github.com/cgohlke/geospatial-wheels)

## Deployment

1. Place the python script inside a folder of .kmz files you wish to convert.
2. Run the script and see the converted GeoTIFFs get placed in a folder titled "Converted GeoTIFFs.
3. (A conversion log is also created and updated each time the script is run that outputs a list of changes)
